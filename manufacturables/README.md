\# Manufacturables

This is a collection of all of the parts that you are going to need to be able to make the project. I will breakdown here how many of each part you are going to require.



For all parts, use PETG filament with tree supports.



\## elementa\_ushape

This is the first half of the ushape that the payload attachs to, you need one.



\## elementb\_ushape

This is the second half of the ushape that the payload attachs to, you need one.



\## base\_manufacturable

This is the main base peice for the project, make sure you have supports for the weird bearing hole in the middle, you need one



\## elementa\_manufacturing\_armelement

This is one half of one of the arms, use tree supports for everything (not "only touching buildplate!!!). You need two of this.



\## elementb\_manufacturing\_armelement

This is the other half of one of the arms, use tree supports for everything (not "only touching buildplate!!!). You also will need two of this.



\# footv1

This is the foot for the base, one of the many mounting options. you need 4 of these and no supports. print on top face.



\# foot cap v1

These are bolt hole covers for the feet for more surface area on the ground, you also need 4 of these.



\# cannon-mount

this is the mount for the cannon payload. its a very complex print, you may want to find a good printer to print it on. Use tree supports not only on buildplate, slower printing speed with partend fan always at 100% (set it in your slicer). you only need one of these.



\# jointbase

this is another complex one, use tree supports also NOT only on buildplate, ensure the pulley is well supported on all faces that are overhang, and you only need one of these.
